## 关于代码高亮  
\`\`\` 后面跟上语言如 php、js等  如 `js`  
但是有时 `php` 在 Atom 上并不会高亮，这时在首航添加 `<?php` 即可  
```php
<?php
/**
*
*/
class ClassName extends AnotherClass
{

	function __construct(argument)
	{
		# code...
	}
}
```
